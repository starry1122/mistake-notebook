环境配置（有问题可以搜CSDN）：

1.IDEA的安装：淘宝搜一个IDEA，安装，几块钱，附带教程（支持远程安装）。JDK安装并且配置到IDEA当中，用的版本是JDK17.

2.maven配置：CSDN搜教程

3.Tesseract 配置：

**1）安装到 D 盘**

进入https://github.com/UB-Mannheim/tesseract/wiki安装，如果进不去我可以给你安装包

- 安装目录设为：`D:\Tools\Tesseract-OCR`
- 安装时尽量勾选语言包（至少 `eng`，最好同时有 `chi_sim`）

**2）确认目录结构**
安装后至少应有：

- `D:\Tools\Tesseract-OCR\tesseract.exe`
- `D:\Tools\Tesseract-OCR\tessdata\`

**3）补中文语言包（如果没有）**

- 进入https://github.com/tesseract-ocr/tessdata_fast/raw/main/chi_sim.traineddata下载 `chi_sim.traineddata`
- 放到：`D:\Tools\Tesseract-OCR\tessdata\chi_sim.traineddata`
- 同时确认 `eng.traineddata` 也在这个目录

**4）配置环境变量 PATH（可选但推荐）**

- 把 `D:\Tools\Tesseract-OCR` 加入系统或用户 `Path`
- 新开一个 PowerShell 窗口

**5）验证 Tesseract 是否可用**
在新 PowerShell 执行：

```powershell
tesseract --version
tesseract --list-langs
```

预期：

- 第一条有版本信息
- 第二条里有 `chi_sim` 和 `eng`

> 如果你不想配 PATH，也可以跳过这一步，后面直接用绝对路径配置。

**6）修改你项目配置（重点）**
编辑：`D:/Java_project/mistake_notebook/back/src/main/resources/application.properties`  
写成下面这样（D盘绝对路径版）：

```properties
mistake.upload-dir=uploads
mistake.ocr.mode=tesseract
mistake.ocr.tesseract-cmd=D:/Tools/Tesseract-OCR/tesseract.exe
mistake.ocr.lang=chi_sim+eng
```

**7）重启后端**

**8）先做命令行识别测试（建议）**
准备一张图片：`D:\test\q1.png`，执行：

```powershell
D:\Tools\Tesseract-OCR\tesseract.exe "D:\test\q1.png" stdout -l chi_sim+eng
```

能看到识别文本，说明本地OCR环境OK。

**9）在系统里验证上传**

- 前端上传错题图片
- 结果会是：
  - 成功：`ocr_status=success`，`ocr_text` 含“本地Tesseract识别结果（可能存在误差）”
  - 失败：`ocr_status=failed`，`ocr_text` 含“tesseract失败原因 + mock回退文本”

**10）常见报错排查**

- `CreateProcess error=2`：路径写错，检查 `mistake.ocr.tesseract-cmd`
- `Failed loading language 'chi_sim'`：`chi_sim.traineddata` 不在 `tessdata` 或损坏
- `Error opening data file`：检查 `D:\Tools\Tesseract-OCR\tessdata` 是否存在
- 全是乱码/空文本：图片质量问题（倾斜、模糊、噪点），先提升清晰度

4.MySQL说明：MySQL安装教程搜CSDN，在mistake_notebook\back\src\main\resources\application.properties配置你的用户名和密码

```

spring.datasource.password=123456
```

5.项目导入：项目导入IDEA的时候，要把.idea删除重新配置；然后把maven环境reload一下；

6.运行，先通过IDEA开启后端，再通过命令行开启前端

前端开启步骤：

①Win+R输入cmd，打开命令行

②输入D:，然后输入cd D:\Java_project\mistake_notebook\front进入前端目录

③输入npm run dev（npm install）
